---
description: Push changes to GitHub
---

This workflow will help you push your changes to the GitHub repository.

1. Check status
```bash
git status
```

2. Add all changes (if any)
```bash
git add .
```

3. Commit changes (if any)
```bash
git commit -m "Update code"
```

4. Push to remote
// turbo
```bash
git push origin main
```
